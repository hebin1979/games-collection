# 🎮 小游戏合集

经典棋牌游戏网页版，随时畅玩无需下载。

## 🕹️ 游戏列表

### 🃏 21点
经典 Blackjack 游戏，与庄家一对一对决。

### ♠️♥️ 德州扑克
经典德州扑克游戏，与 AI 对战。

### 💎 宝石消除
经典三消游戏，挑战高分。

## 🚀 使用方法

直接在浏览器中打开 HTML 文件即可开始游戏：
- `index.html` - 游戏首页
- `blackjack.html` - 21点游戏
- `texas-holdom.html` - 德州扑克
- `match3.html` - 宝石消除

## 📱 兼容性

支持电脑和手机浏览器访问。
